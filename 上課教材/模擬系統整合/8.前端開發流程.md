前端開發流程
---

1. 建立專案目錄結構
- uv init
- uv sync
- uv add pyedb pywebview 
- npm create vite@latest


2. 告訴AI，開發Pywebview GUI介面，開發階段用typescript編寫，並借助Vite渲染框架，但是最終要轉成純html+css+js的前端頁面，並且要能和後端python溝通。將上面開發規範寫入gemini.md。

3. 開發階段，建立GUI並確認與後端python溝通正常。

3. 編寫README，包含專案介紹、安裝說明、使用說明、開發說明等內容。

3. 需要產生run.bat來啟動專案，如果uv不存在，則下載並安裝uv。同時檢查python環境是否存在，若不存在則自動下載並安裝python及uv sync所需的相關套件。最後啟動uv run 啟動GUI。

4. 發布階段，需要將typescript轉成純html+css+js，並且將前端頁面和後端python打包成zip壓縮檔，方便分發和部署。
