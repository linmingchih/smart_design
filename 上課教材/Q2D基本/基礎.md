Q2D 基礎
---
### 類型(Solution Type)：
- Open：如微帶線等，外部無金屬包覆，內部有空氣或介質。
- Closed：如電纜等，外部有金屬包覆，內部有空氣或介質。

### Q2D編輯器

1. 使用者可以在 Q2D 編輯器中建立傳輸結構的幾何圖形，並定義圖形的屬性，如材料及電氣特性，以便進行傳輸結構的模擬。提取阻抗和速度等。
2. 材料主要分成兩類：導體和介質。
    - 導體的電氣特性主要由電導率（σ）決定，
    - 而介質的電氣特性則由相對介電常數（εr）決定。

3. 結構當中必須包含Signal與Reference Ground。
    - Signal是傳輸線的主體，負責傳輸電磁波。
    - Reference Ground則提供一個穩定的參考點，確保信號的正確傳輸和電磁場的分布。
3. 編輯器背景默認為真空，使用者也可以繪製真空幾何包覆傳輸線金屬結構，好處是可以在幾何結構上觀察電磁場分布。

### 設定
1. 金屬種類
    - Signal Line
    - Non Ideal Ground
    - Reference Ground: 
    - Floating Line
    - Surface Ground (Closed Type Only)
2. 邊界條件
    - Finite Conductivity

3. 設定組合
    - 單一導體：不可解析，因為沒有參考地，無法形成傳輸線。
    - 雙導體：可解析，形成傳輸線。
        - Signal Line + Reference Ground
        - Non Ideal Ground + Reference Ground
    - 三導體：可解析，形成傳輸線。
        - Signal Line + Reference Ground + Floating Line
        - Non Ideal Ground + Reference Ground + Floating Line
        - Surface Ground + Reference Ground + Floating Line
