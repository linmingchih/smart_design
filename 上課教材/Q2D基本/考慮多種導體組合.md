考慮多種導體組合
---
在Q2D當中，最小可以模擬的傳輸結構是由兩種導體組成的結構，金屬種類有以下幾種：
- Reference Ground：Open模式底下，必須存在且唯一 
- Signal Line：可以存在且多個
- Non Ideal Ground：可以存在且多個。與Signal Line僅語意不同，求解結果完全相同
- Floating Line：在存在至少一個Signal Line或Non Ideal Ground的情況下，可以存在且多個。
- Surface Ground：Closed模式底下，必須存在且唯一

嚴格來說，僅有三大類，
- Reference Ground, Surface Ground：不納入矩陣計算
- Signal Line, Non Ideal Ground：納入矩陣計算
- Floating Line：不納入矩陣計算

### 雙導體結構的組合
一個好奇的問題是，那些結構的組合是可以解析的呢？為了驗證這個問題，我們可以嘗試模擬所有兩種導體組合的結構，看看哪些組合是可以成功求解的。

1. `ReferenceGround + SignalLine`
2. `ReferenceGround + NonIdealGround`

程式碼如下：
```python
from ansys.aedt.core import Q2d
from itertools import combinations_with_replacement

result = {}
cond_type = ["SignalLine", "ReferenceGround", "NonIdealGround", "FloatingLine"]
for a, b in combinations_with_replacement(cond_type, 2):
    q2d = Q2d(version='2025.2', design = f'{a}_{b}')
    q2d.modeler.units = "mm"
    q2d.solution_type = "Open"
    b1 = q2d.modeler.create_rectangle([0, 0, 0], [1, 1, 1], name='b1')
    b2 = q2d.modeler.create_rectangle([5, 0, 0], [1, 1, 1], name='b2')

    q2d.auto_assign_conductors()
    setup = q2d.create_setup() 

# "SignalLine", "ReferenceGround", "NonIdealGround", "FloatingLine"
    
    if (q2d.toggle_conductor_type('b1', new_type=a)
    and q2d.toggle_conductor_type('b2', new_type=b)
    and q2d.analyze()):
        result[f'{a}_{b}']=True
    else:
        result[f'{a}_{b}']=False
```

### 三導體結構的組合
以下為允許的三導體類型組合（不區分順序，可重複）：

1. `ReferenceGround + SignalLine + SignalLine`
2. `ReferenceGround + SignalLine + NonIdealGround`
3. `ReferenceGround + SignalLine + FloatingLine`
4. `ReferenceGround + NonIdealGround + NonIdealGround`
5. `ReferenceGround + NonIdealGround + FloatingLine`

```python
from ansys.aedt.core import Q2d
from itertools import combinations_with_replacement

result = {}

cond_type = ["SignalLine", "ReferenceGround", "NonIdealGround", "FloatingLine"]

    b3 = q2d.modeler.create_rectangle([10, 0, 0], [1, 1, 1], name='b3')

    q2d.auto_assign_conductors()

# "SignalLine", "ReferenceGround", "NonIdealGround", "FloatingLine"
```

### Closed條件


```python
from ansys.aedt.core import Q2d
from itertools import combinations_with_replacement

status = {}
result = {}

# "ReferenceGround", "SignalLine", "NonIdealGround", "FloatingLine"
cond_type = ["SignalLine", "NonIdealGround", "FloatingLine"]
for a, b, c in combinations_with_replacement(cond_type, 3):
    q2d = Q2d(version='2025.2', design = f'{a}_{b}_{c}')
    q2d.modeler.units = "mm"
    q2d.solution_type = "Closed"

    
    b1 = q2d.modeler.create_rectangle([-5, 0, 0], [1, 1, 1], name='b1')
    b2 = q2d.modeler.create_rectangle([0, 0, 0], [1, 1, 1], name='b2')
    b3 = q2d.modeler.create_rectangle([5, 0, 0], [1, 1, 1], name='b3')

    q2d.auto_assign_conductors()
    setup = q2d.create_setup() 

# "SignalLine", "ReferenceGround", "NonIdealGround", "FloatingLine"
    
    b0 = q2d.modeler.create_rectangle([-10, -5, 0], [20, 10, 0], name='b0', material='vacuum')
    oModule = q2d.odesign.GetModule("BoundarySetup")
    oModule.AssignSingleSurfaceGround(
    	[
    		"NAME:b0",
    		"Objects:="		, ["b0"],
    		"SolveOption:="		, "SolveInside",
    		"Thickness:="		, "-1000mm"
    	])    


    if (q2d.toggle_conductor_type('b1', new_type=a)
    and q2d.toggle_conductor_type('b2', new_type=b)
    and q2d.toggle_conductor_type('b3', new_type=c)
    and q2d.analyze()):
        status[f'{a}_{b}_{c}'] = True
        q2d.export_matrix_data(f'd:/demo/{a}_{b}_{c}.csv', problem_type="CG, RL")

        
    else:
        status[f'{a}_{b}_{c}'] = False
        q2d.delete_design()
        
```
