# 讓第一個範例跑起來

## 綱要

1. **建立新目錄**
   - 為新的測試或專案建立一個專屬的資料夾，保持環境乾淨。
   - 範例：建立一個名為 `pymechanical_test` 的資料夾。

2. **使用 `uv` 初始化與套件管理**
   - `uv init --python 3.10`：初始化專案環境，並指定使用 Python 3.10 版本。
   - `uv sync`：同步與確保環境依賴一致。
   - `uv add <想要學習的模組>`：安裝並加入你想要學習的目標模組。
     - **以 Ansys Mechanical 為例：**
       ```bash
       uv add ansys-mechanical-core
       ```
     - *(註：若使用傳統 pip，對應的指令為 `pip install ansys-mechanical-core`)*

3. **在模組當中找一個範例**
   - 從官方文件或 GitHub repository 中尋找適合初學者的範例程式碼。
   - **本次實作範例：Basic valve implementation (基本閥門實作)**
     - 參考網址：[Basic valve implementation](https://mechanical.docs.pyansys.com/version/stable/examples/gallery_examples/01_basic/valve.html)
     - **範例簡介：** 這個範例示範了如何使用 PyMechanical 對閥門幾何模型進行靜態結構分析。內容涵蓋匯入幾何模型、指定材料、生成網格、設定邊界條件（固定支撐、無摩擦支撐和壓力），以及視覺化變形與應力結果。

4. **逐步執行，完整執行範例**
   - 觀察程式碼結構，並將官方提供的 Python 腳本下載或複製到目錄中。
   - 逐步執行（或分段執行）以理解每個步驟（例如：啟動 Mechanical、匯入網格、設定邊界條件）的作用。
   - 確保範例能從頭到尾順利跑完並產生預期結果。
