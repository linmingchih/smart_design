# 第 5 章 建立操作介面：為 Python 腳本打造現代化 Web GUI

當 Python 自動化腳本開發與測試完成後，為了讓不熟悉程式碼的工程師也能輕鬆使用，我們需要為其建立一個直覺、美觀的操作介面（GUI）。傳統的 Python GUI 庫（如 Tkinter 或 PyQt）在介面美觀度與現代化體驗上存在局限，因此在業界的最新實踐中，我們通常會採用 **Web-based 的前後端分離架構**。

本章將帶領你利用前端的 **React (TypeScript)** 與後端的 **FastAPI**，並透過 AI 的協助，快速為你的 Python 腳本搭建專屬的操作介面，最後利用 `run.bat` 實現一鍵啟動。

---

## 1. 建立前端專案：React 與 TypeScript

我們將使用 Vite 作為前端建置工具，它擁有極快的啟動與熱更新速度。透過它，我們可以快速初始化一個基於 React 與 TypeScript 的專案。

打開終端機（Terminal）並輸入以下指令來建立名為 `frontend` 的專案：

```bash
# 初始化 Vite 專案
npm create vite@latest
```

這樣我們就準備好了一個現代化且具備型別檢查（TypeScript）的前端開發環境。

---

## 2. 建立後端環境：導入 FastAPI

我們需要一個「橋樑」來接收前端網頁傳來的參數，並驅動背後的 Python 模擬腳本，這個橋樑就是 Web API。我們將使用目前 Python 生態系中最輕量、極速的框架：**FastAPI**。

使用現代化 Python 套件管理工具 `uv` 來安裝 FastAPI 與伺服器套件：

```bash
# 加入 FastAPI 與 Uvicorn
uv add fastapi uvicorn
```

透過 FastAPI，你可以輕鬆寫出接收 HTTP Request 的端點（Endpoints），並將其連接到你先前的自動化函數中。

---

## 3. 讓 AI 幫你寫 UI：自動拉取參數與生成網頁

這一步是展現「AI 輔助開發」強大威力的時刻。你不必從頭手刻複雜的表單元件，只需將後端的 Python 腳本提供給 AI Agent（如 ChatGPT 或 Gemini），讓它幫你生成前端的 React 程式碼。

**你可以這樣下 Prompt（提示詞）：**

> 「請閱讀我以下的 Python 模擬腳本，提取出所有需要使用者輸入的變數與參數（例如材質、頻率、幾何尺寸等）。接著，利用 React 與 TypeScript（搭配 Tailwind CSS，如果有的話）幫我生成一個對應的網頁表單介面。當使用者按下『執行模擬』按鈕時，請透過 Fetch API 以 JSON 格式將這些參數 POST 到 `http://localhost:8000/run_simulation` 的 FastAPI 端點。」

AI 會幫你完成：
1. **參數解析**：自動識別 Python 程式碼中 `def run_simulation(length=10, width=5, material="copper"):` 的參數。
2. **表單建立**：在 React 中生成對應的輸入框（Input）、下拉式選單（Select）以及狀態管理（`useState`）。
3. **串接邏輯**：寫好向後端發送 HTTP 請求的網路通訊程式碼。

你只需將 AI 產生的程式碼貼入 `frontend/src/App.tsx` 中進行微調即可。

---

## 4. 整合與一鍵啟動：建立 `run.bat`

現在我們有了負責介面的 React 前端（通常運行在 5173 port）以及負責驅動 Python 的 FastAPI 後端（通常運行在 8000 port）。為了讓使用者無需懂得如何下指令，我們可以寫一個批次檔（Batch script）來一鍵啟動兩邊的伺服器，並自動打開瀏覽器。

在專案的根目錄下建立一個名為 `run.bat` 的檔案，內容如下：

```bat
@echo off
chcp 65001 >nul
echo =========================================
echo       啟動 AI 模擬自動化操作介面...
echo =========================================

:: 啟動 FastAPI 後端伺服器 (在新視窗)
echo [1/3] 正在啟動 FastAPI 後端...
start "Backend Server" cmd /c "uv run uvicorn main:app --reload --port 8000"

:: 啟動 React 前端開發伺服器 (在新視窗)
echo [2/3] 正在啟動 React 前端...
start "Frontend Server" cmd /c "cd frontend && npm run dev"

:: 等待幾秒鐘確保伺服器皆已啟動
echo [3/3] 正在開啟瀏覽器...
timeout /t 3 >nul

:: 開啟預設瀏覽器並導向前端網址
start http://localhost:5173

echo 啟動完成！請在瀏覽器中操作介面。
pause
```

### 使用方式：
使用者只需要雙擊執行 `run.bat`，系統就會自動：
1. 叫起背景的 Python FastAPI 服務。
2. 叫起 Vite/React 前端服務。
3. 自動打開預設瀏覽器進入 GUI 畫面。

透過這個流程，我們完美地將生硬的 Python 自動化腳本，包裝成了一個具備現代化 UI、容易佈署與使用的 Web 應用程式。
